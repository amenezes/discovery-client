# discovery-client

Async Python client for [consul](https://consul.io).

HTTP engine options available:

- aiohttp `default`;
- httpx.

## Installing

Install and update using pip:

### default client

````bash
pip install -U discovery-client
````

### httpx client

````bash
pip install -U 'discovery-client[httpx]'
````
